# 🛒 AMAZON SALES DASHBOARD

Welcome to the Amazon Sales Dashboard project!

This project showcases an interactive Amazon Sales Dashboard built in Power BI to analyze sales performance, profit trends, product categories, and quarterly business insights. The dashboard helps businesses track important KPIs and make data-driven business decisions.

## 🚀 Features

📈 Interactive charts and KPI visuals

📊 Sales and Profit analysis by category

🛍️ Product category performance tracking

📅 Quarterly sales and profit insights

🗺️ State-wise filtering and analysis

🎯 Dynamic slicers for better data exploration

📦 Category-wise comparison dashboards

📂 Clean and structured dataset integration

## 🛠️ Tech Stack

Power BI – Dashboard creation & visualization

Microsoft Excel / CSV – Data source

DAX – Measures & calculations

Power Query – Data cleaning & transformation

## 📈 Dashboard Analysis

The dashboard provides detailed profit insights across different product categories.

🛒 Sales Analysis
